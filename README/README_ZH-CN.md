## 功能
- **帖子下载**：使用URL轻松下载特定Kemono.cr帖子中的文件。
- **创作者存档**：一键批量下载创作者档案中的所有帖子和文件。
- **文件类型支持**：处理图像（JPG、PNG、GIF）、视频（MP4）、存档（ZIP、7Z）、PDF等。
- **并发下载**：调整同时下载数量（1-10）以获得最佳性能。
- **文件去重**：使用URL哈希防止冗余下载。
- **图像预览**：在下载前预览图像以验证内容。
- **详细日志**：通过应用内控制台跟踪进度和排除故障。
- **跨平台UI**：使用PyQt6构建现代直观的界面，兼容多个操作系统。
- **媒体播放**：使用内置播放控件（播放、暂停、搜索、音量）预览视频和GIF。
- **多语言支持**：在英语、日语、韩语和简体中文之间动态切换。
- **自动更新**：启动时检查新版本，可选择通知。

## 安装
Kemono Downloader现在使用[Briefcase](https://briefcase.readthedocs.io/)打包，使其更容易作为本地应用程序在各平台上运行或分发。您可以从源代码构建或使用预编译的二进制文件（如果可用）。

### 预编译二进制文件 - **.EXE文件**
预编译二进制文件通过**Ko-fi**上的贡献提供。如果您想要即用型可执行文件的便利性而不是从源代码构建，请在这里支持项目：**[在Ko-fi获取二进制文件](https://ko-fi.com/izeno/shop)**  

#### 可用二进制文件
- **Windows**:  
  - **便携版**: `W-KemonoDownloader-x86_64-Portable.exe`  
  - **安装程序**: `W-KemonoDownloader-x86_64.msi`  
- **macOS**:  
  - `M-KemonoDownloader-arm64.dmg` (Apple Silicon)  
  - `M-KemonoDownloader-x86_64.dmg` (Intel)  
- **Linux**:  
  - `L-KemonoDownloader-x86_64.pkg.tar.zst` (基于Arch)  
  - `L-KemonoDownloader-x86_64.rpm` (基于Fedora/Red Hat)  
  - `L-KemonoDownloader-amd64.deb` (基于Debian/Ubuntu)  

支持持续开发并在此处获得访问：**[Ko-fi二进制文件页面](https://ko-fi.com/izeno/shop)**

### 使用Briefcase构建（所有平台）
1. 确保您的系统安装了**Python 3.9+**（Windows、macOS、Linux）。
2. 克隆此存储库：
   ```bash
   git clone https://github.com/VoxDroid/KemonoDownloader.git
   cd KemonoDownloader
   ```
3. 创建并激活虚拟环境：
    ```bash
    python -m venv .venv
    .venv/Scripts/activate # Windows 
    source .venv/bin/activate # Linux
    ```
4. 安装依赖项：
   ```bash
   python -m pip install -r requirements.txt
   ```
5. 构建应用程序：
   - **Windows**: `briefcase build windows`
   - **macOS**: `briefcase build macos`
   - **Linux**: `briefcase build linux`
   - **通用**: `briefcase build` *（推荐）*
6. 运行应用程序：
   - **Windows**: `briefcase run windows`
   - **macOS**: `briefcase run macos`
   - **Linux**: `briefcase run linux`
   - **通用**: `briefcase dev` *（推荐）*
   - **注意**：需要互联网连接才能从Kemono.cr获取内容。

## 使用方法
启动后，您将看到带有"启动"按钮的介绍屏幕。单击它进入主界面，包含四个选项卡：**帖子下载器**、**创作者下载器**、**设置**和**帮助**。应用内帮助选项卡包含全面的用户手册。

### 入门
- 应用程序在指定的保存位置创建默认目录（`Downloads`、`Cache`、`Other Files`）。
- 确保有活跃的互联网连接以访问Kemono.cr内容。
- 浏览帮助选项卡获取详细说明和故障排除提示。

### 帖子下载器选项卡
- **目的**：从单个Kemono.cr帖子下载文件。
- **使用方法**：
  1. 在"输入帖子URL"字段中输入帖子URL（例如：`https://kemono.cr/patreon/user/123456789/post/123456789`）。
  2. 点击"添加到队列"将其添加到列表中。
  3. 点击眼睛图标查看文件，按类型筛选（例如：JPG、ZIP），并选择要下载的文件。
  4. 点击"下载"开始，并通过进度条和控制台监视进度。

### 创作者下载器选项卡
- **目的**：批量下载创作者档案中的内容。
- **使用方法**：
  1. 在"输入创作者URL"字段中输入创作者URL（例如：`https://kemono.cr/patreon/user/123456789`）。
  2. 点击"添加到队列"将其添加到列表中。
  3. 点击眼睛图标获取帖子，配置选项（主文件、附件、内容图像），并选择帖子。
  4. 点击"下载"开始，并通过界面跟踪进度。

### 设置选项卡
- **目的**：自定义应用程序的行为。
- **使用方法**：
  1. 设置下载的文件夹名称和保存目录。
  2. 使用滑块或旋转框调整同时下载数量（1-10）。
  3. 点击"应用更改"保存。

### 帮助选项卡
- **目的**：访问嵌入式用户手册。
- **使用方法**：导航到帮助选项卡阅读详细指南、示例和支持信息。
